UniPulse Project Documentation

Project Name: UniPulse - Campus Blood Donor Database

Date: 2025-12-13

Author: UniPulse Team

1. Project Overview

UniPulse is a web-based platform designed to connect blood donors with recipients within a university campus. It features real-time blood inventory tracking, donor registration, hospital request portals, and emergency alerts.

2. Technology Stack

Frontend (User Interface)

  • HTML5: Semantic structure of the web pages.
  • Vanilla CSS: Custom styling without external frameworks to ensure a unique, premium look.
  • Google Fonts: 'Outfit' font for modern typography.

Backend (Server Logic)

  • Native PHP: Server-side logic.
  • Data Simulation: PHP arrays to simulate database data.
